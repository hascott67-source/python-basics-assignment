1. types_and_vars.py
Description
Stores your name, age, and height in variables

Prints an introduction sentence

Calculates your age in 5 years

Computes the area of a rectangle

Demonstrates arithmetic operators and string repetition

Example Output:
Hello, my name is Henry. I am 57 years old and 1.8 meters tall.
In 5 years, I will be 62 years old.
The area of a 5.5 x 2 rectangle is 11.0.
Age + 10 = 67
Age % 4 = 1
Repeating my name 3 times: HenryHenryHenry

2. simple_calculator.py
Description
Asks the user for two numbers

Asks for an operation (+, -, *, /)

Performs the calculation

Prints the result

Handles basic invalid input

Example Output:
Enter the first number: 7
Enter the second number: 3
Choose an operation (+, -, *, /): *
7.0 * 3.0 = 21.0

3. string_fun.py
Description
Prompts the user for a word

Prints the length

Prints the word in uppercase

Prints the word repeated 3 times (same line and separate lines)

Example Output:
Enter a word: cloud
Length of the word: 5
Uppercase: CLOUD
Repeated 3 times (same line): cloudcloudcloud
Repeated 3 times (separate lines):
cloud
cloud
cloud

Run any script using:
python filename.py
Example:
python types_and_vars.py
